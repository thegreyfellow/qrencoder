# qrencoder

An attempt to make a customizable QR code generator, using Crystal.

## Installation

- setup dependencies:

`$ shards install`

- generate binary:

`$ crystal build --release src/qrencoder.cr`

## Usage

- use executable:

`$ ./qrencoder "hello world"`
